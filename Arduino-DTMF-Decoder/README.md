# Arduino DTMF Decoder

Video will add soon.

<br>

<b>Part list:</b>
<ul>
   <li>Arduino Pro Micro 32U4</li>
   <li>MT8870 DTMF</li>
   <li>3.5 mm Male to Male Stereo Audio Aux Cable</li>
   <li>USB to Micro USB cable</li>
</ul>

<b>Pin Connections:</b>
Arduino > Decoder
<ul>
    <li>VCC > VCC</li>
    <li>GND > GND</li>
    <li>D8 > Q1</li>
    <li>D7 > Q2</li>
    <li>D6 > Q3</li>
    <li>D5 > Q4</li>
    <li>D3 > STD</li>
</ul>
